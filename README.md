# 🍋 Little Lemon Restaurant

This is a simple front-end web app for the Little Lemon Restaurant.  
It was built using **React**, **Vite** and **React Router** .

---

## 📌 Pages Included

- 🏠 Home
- 📅 Reservation
- 👤 Profile
- ✅ Confirmation

## 🛠 Built With

- React Router
- CSS
